# CopyCat 架构设计

## 概述

CopyCat 是一个专业的 macOS 本地数据库管理应用，采用 Swift + SwiftUI 构建。

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        UI 层 (SwiftUI)                        │

│  │      View       │  │                  │  │  Editor  │  │
│  └────────┬─────────┘  └────────┬─────────┘  └─────┬────┘  │
└───────────┼────────────────────────┼───────────────────┼───────┘
            │                        │                   │

│  │   ViewModels     │  │   State Mgmt     │  │  Actions │  │
```

## 模块职责

### App 层
- `CopyCatApp.swift`: 应用主入口
- `AppDelegate.swift`: AppKit 桥接

### Core 层
- **Database**: 数据库连接管理、查询执行
- **Security**: 密码加密存储、Keychain 管理
- **Models**: 数据模型定义

### UI 层
- **Scenes**: 页面级视图
- **Components**: 可复用组件
- **Resources**: 资源文件

### Utilities 层
- 日志工具
- 错误处理
- 扩展

## 数据流

```
User Action → View → ViewModel → Core Layer → View Update
```

## 关键设计决策

1. **纯函数化视图**: View 不包含业务逻辑
2. **状态管理**: ViewModel 作为单数据源
3. **并发安全**: 使用 `@MainActor` 保证 UI 更新安全
4. **依赖注入**: 通过 `@EnvironmentObject` 传递服务
