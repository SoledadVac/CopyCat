# CopyCat v0.1.0

## 发布日期
2026-03-03

## 新功能

### 数据库连接管理
- ✅ 支持多种数据库类型
  - MySQL
  - MariaDB
  - PostgreSQL
  - SQL Server
  - Oracle
  - SQLite
  - MongoDB
  - Redis

- ✅ 连接配置功能
  - SSH 隧道支持
  - SSL 加密连接
  - 连接分组管理
  - 颜色标签标记
  - 连接测试功能

- ✅ 安全存储
  - Keychain 加密存储密码
  - AES-256-GCM 加密
  - PBKDF2-SHA256 密钥派生

### 查询编辑器
- ✅ SQL 查询编辑器
  - 代码高亮（待实现）
  - 查询历史记录
  - 查询执行时间统计
  - 结果行数显示

- ✅ 安全操作
  - 危险操作二次确认
  - 删除/截断/删除表操作防护

### 数据管理
- ✅ 表结构查看
  - 字段信息显示
  - 索引信息
  - 外键关系

- ✅ 数据操作（部分实现）
  - 数据导入
  - 数据导出（待实现）
  - 备份功能（待实现）
  - 数据生成器（待实现）

### 用户界面
- ✅ macOS 原生设计
  - SwiftUI 构建
  - 符合 Human Interface Guidelines
  - 深色模式支持
  - 响应式布局

- ✅ 连接卡片
  - 数据库类型图标
  - 连接状态显示
  - 最后连接时间
  - 右键上下文菜单
