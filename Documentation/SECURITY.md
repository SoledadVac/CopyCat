# 安全规范

## 概述

CopyCat 严格遵循企业级安全标准，确保用户数据安全。

## 密码安全

### 加密算法
- **主密钥**: 32字节随机密钥，存储在 Keychain
- **派生密钥**: PBKDF2-SHA256 加盐派生
- **加密**: AES-256-GCM 加密存储

### 存储位置
- 连接密码: Keychain (系统级安全存储)
- 加密盐值: Keychain
- 主密钥: Keychain

### 审计日志
所有密码操作记录 OSLog 日志:
- 保存密码
- 获取密码
- 删除密码

## SQL 注入防护

### 参数化查询
所有数据库查询使用参数化查询:
```swift
try executor.execute(
    "SELECT * FROM users WHERE id = ?", 
    parameters: [userId]
)
```

### 输入验证
- 禁止直接拼接 SQL 字符串
- 所有用户输入必须参数化
- 查询前进行语法检查

## 沙盒权限

### 最小权限原则
仅申请必要的权限:
- `com.apple.security.files.user-selected.read-write`: 用户选择文件读写

### 文件访问
- 仅访问用户显式选择的文件
- 不后台扫描文件系统
- 不访问系统敏感目录

## 隐私保护

### 数据不上传
- 所有数据本地处理
- 无网络数据发送
- 无远程服务器连接

### 隐私清单
PrivacyInfo.xcprivacy 声明:
- 数据收集: 无
- 数据使用: 本地数据库管理
- 第三方共享: 无

## 审计日志

### 记录内容
- 所有数据修改操作
- 连接建立/断开
- 密码操作

### 日志级别
- Debug: 调试信息
- Info: 普通操作
- Error: 错误信息
- Fault: 严重错误

### 日志脱敏
- 密码字段不记录
- 敏感数据掩码显示
- 仅记录操作类型和时间
