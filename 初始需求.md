

# 🗃️ 核心功能深度解析（CopyCat 功能介绍）



## 🌐 一、全栈数据库连接管理（核心基石）
| 功能 | 说明 | 专业价值 |
|------|------|----------|
| **多数据库支持** | MySQL/MariaDB/Oracle/SQL Server/PostgreSQL/SQLite/MongoDB/Redis | 一套工具管理全技术栈，避免切换成本 |
| **连接配置** | SSH/SSL/HTTP 隧道、云数据库直连（阿里云RDS/AWS RDS）、私有网络配置 | 安全访问生产环境，符合企业安全规范 |
| **连接分组** | 自定义文件夹分组（按项目/环境/客户）、颜色标签标记 | 百级连接高效管理，避免误操作 |
| **连接测试** | 一键测试连接状态 + 详细错误日志定位 | 运维效率提升50% |

## 📊 二、对象管理与可视化设计（开发者日常高频）
| 模块 | 核心能力 | 实战场景 |
|------|------------|----------|
| **表设计器** | 可视化建表（字段/索引/外键）、DDL 实时预览、结构对比同步 | 新人快速建表，避免手写SQL错误 |
| **ER 图** | 自动反向生成关系图、拖拽调整布局、导出高清 PNG/PDF | 需求评审/架构文档必备 |
| **查询构建器** | 拖拽式生成 SELECT 语句（无需写 JOIN/WHERE） | 业务人员自助查数据 |
| **数据编辑** | 表格视图直接增删改（带事务回滚）、BLOB/JSON 可视化编辑 | 快速修复测试数据 |
| **对象搜索** | 全局搜索表/字段/存储过程（支持正则） | 遗留系统维护神器 |

## ⚡ 三、数据操作与迁移（DBA 核心场景）
| 功能 | 亮点 | 企业级价值 |
|------|------|------------|
| **数据传输** | 跨数据库迁移（MySQL→PostgreSQL）、增量同步、冲突处理策略 | 异构数据库迁移零代码 |
| **数据同步** | 定时双向同步（主从/灾备）、差异对比、冲突解决 | 保障数据一致性 |
| **导入导出** | 支持 CSV/Excel/JSON/XML/SQL，自定义分隔符/编码/格式 | 与业务系统无缝对接 |
| **备份还原** | 一键压缩备份、计划任务自动备份、时间点恢复（PITR） | 满足等保2.0备份要求 |
| **数据生成器** | 模拟生成测试数据（姓名/手机号/地址等20+模板） | 测试环境快速搭建 |
